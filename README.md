# LLM Playground

A flexible playground for experimenting with multiple Large Language Models (LLMs) and exploring various use cases.

## Overview

This repository serves as an experimental workspace for working with different LLM providers and testing various AI-powered applications and workflows.

## Setup

1. Copy the environment template to create your local configuration:
   ```bash
   cp .env.template .env
   ```

2. Edit [.env](.env) and add your API keys and configuration values

3. Source the environment variables:
   ```bash
   source .env
   ```

## Usage

Each subdirectory may contain different experiments and use cases. Explore the folders to see what's available.
